Creating lists

Html tags

Tag Name Description
<ul> Unordered list A list in a non specific order
<ol> Ordered list A list in a non specific order
<li> List tag The list item

CSS list properties

The <list-style-type> property specifies the type of list item marker

Property List type Description
<default> unordered An unordered list with bullet points
<none> unordered Renders without bullet points
<circle> unordered An unordered list with circle bullet points
<square> unordered An unordered list with square bullet points
<default> ordered An ordered list with numerical markers
<none> ordered Renders without numbered points
<upper-roman> ordered An ordered list with Roman Numeral markers
<lower-alpha> ordered An ordered list with lowercase latin alphabet markers
